====PHPocX 4.1 by 2mdc.com====
http://www.phpdocx.com/

====Requirements====
Phpdocx needs ZipArchive and XSLT on PHP5. If you are not sure if is installed in your server run check.php or contact with your Hosting provider.

====How to Install====
Use your favorite FTP client to connect to your server.

- Create a folder anywhere called phpdocx (or other name) accessible from the web server.

- Drag and drop all the documents inside the folder you created in your server.

- Include your license key in the config/phpdocxconfig.ini file (you should also include the domain in the config/phpdocxconfig.ini file for the CORPORATE version)

You don't need special permissions for Phpdocx but writting permissions on the folder you are copying the resulting docx file. If you need to check if PHPDocX is working, point your web browser to one of files of the folder Examples included.
